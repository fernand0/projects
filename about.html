Sobre este sitio

Este sitio recopila diversos proyectos que he ido programando mientras aprendía y experimentaba con diferentes ideas. No son productos comerciales, sino pruebas y ejercicios prácticos.

Mi enfoque

Me gusta aprender haciendo y practicar. Cuando encuentro una idea nueva o interesante, creo pequeños proyectos para entender cómo funciona y tener una prueba de concepto. La mayoría están desarrollados en Python y usan herramientas open source.

Qué hay aquí

Bots y automatización

botElectrico surgió para consultar datos de Red Eléctrica y publicarlos en redes sociales. Los Errbot Plugins fueron mis pinitos con el framework de chatbot Errbot—hay 17 plugins que prueban diferentes integraciones (calendarios, sensores, cámaras web, etc.).

Herramientas CLI

Me divierto creando herramientas de línea de comandos. manage-agenda experimenta con LLMs para extraer eventos de emails, news_manager prueba la API de Google Gemini, y manage-imap es un ejercicio de gestión de correo IMAP.

Redes sociales

socialModules empezó como una forma de publicar en varias redes desde un mismo sitio. linkblog fue otro experimento para agregar enlaces de Twitter automáticamente.

El comienzo

Todo empezó con vimblog.vim, un plugin de Vim para gestionar blogs. Lo descubrí, me gustó tanto que fue el proyecto que me animó a crear mi cuenta de GitHub y empezar a compartir código.

Tecnologías que uso

Principalmente trabajo con:

  • Python: Para casi todo
  • Vim: Mi editor desde hace años
  • Errbot: Para experimentar con chatbots
  • Jekyll: Para este sitio y otros blogs
  • APIs de Google: Calendar, Gemini AI, Gmail
  • LLMs: Gemini, Mistral, Ollama—cuando quiero probar algo con IA

Código abierto

Todo el código está en GitHub. Son proyectos de aprendizaje, así que si ves algo que te sirve o que se puede mejorar, siéntete libre de usarlo o contribuir.

El código no siempre es suficiente para conocer un proyecto de manera adecuada. Por eso he creado este sitio para documentar y explicar cada proyecto.
