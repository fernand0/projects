Sobre este sitio

Este sitio es una colección de mis proyectos de software desarrollados a lo largo de los años. Aquí encontrarás desde bots y automatizaciones hasta herramientas CLI y plugins para diferentes plataformas.

Mi enfoque

Me especializo en crear herramientas que resuelven problemas reales y automatizan tareas repetitivas. La mayoría de mis proyectos están desarrollados en Python y utilizan tecnologías open source.

Algunos proyectos

Automatización y Bots

botElectrico consulta los datos de Red Eléctrica y publica información sobre franjas horarias eléctricas en Twitter, Telegram y Fediverso. La colección de Errbot Plugins incluye 17 plugins para el framework de chatbot Errbot, desde integración con calendarios hasta sensores para Raspberry Pi.

Herramientas CLI

He desarrollado varias herramientas de línea de comandos como manage-agenda (que usa LLMs para extraer eventos de emails y añadirlos a Google Calendar), news_manager (genera artículos con Google Gemini AI), y manage-imap (gestión de buzones IMAP con reglas).

Integración Social

socialModules proporciona módulos para publicar en múltiples redes sociales, y linkblog genera automáticamente posts diarios con enlaces de Twitter.

El comienzo

Mi viaje en GitHub comenzó con vimblog.vim, un plugin de Vim para gestionar blogs. Este proyecto me inspiró a crear mi cuenta de GitHub y compartir mi código con la comunidad open source.

Tecnologías

Las tecnologías que más utilizo en mis proyectos:

  • Python: Lenguaje principal para la mayoría de proyectos
  • Vim: Mi editor de texto preferido
  • Errbot: Framework de chatbot
  • Jekyll: Generador de sitios estáticos
  • Google APIs: Calendar, Gemini AI, Gmail
  • LLMs: Gemini, Mistral, Ollama para procesamiento de lenguaje natural

Código Abierto

Todos mis proyectos son open source y están disponibles en GitHub. Si encuentras algún proyecto útil, siéntete libre de hacer fork, contribuir o simplemente explorar el código.

El código no siempre es suficiente para conocer un proyecto de manera adecuada. Por eso he creado este sitio para documentar y explicar cada proyecto.
