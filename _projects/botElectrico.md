El bot consulta los datos de [REData API](https://www.ree.es/es/apidatos) de Red Eléctrica y publica:

- Cada hora el precio actual durante esa hora y también la hora próxima. Indica el tipo de tramo.
- Al principio de cada tramo horario muestra el precio mínimo y máximo en ese periodo.
- Al principio del día muestra una gráfica donde se indica el momento más barato del día y el más caro
